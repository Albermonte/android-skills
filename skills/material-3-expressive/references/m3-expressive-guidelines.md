# Material 3 Expressive Guidelines (Landing)

Source URL:
https://m3.material.io/

Scope
Top-level Material 3 Expressive guidance and links to core expressive resources.

Key points
- Material Design 3 is Google's open-source design system for building beautiful, usable products.
- M3 Expressive is described as "design with emotion" and focuses on emotion-driven UX.
- Expressive pillars: color, motion, components, typography, and shapes.
- Highlights updated Figma M3 Design Kit, motion physics system, and expanded shape library with shape morph.

Expressive components highlighted
- Toolbars, split buttons, progress indicators, and button groups are featured.
- Split buttons emphasize expressive shape and motion strategies.
- Button groups include shape-shifting buttons that react to each other.
- Links point to the full set of 14 new and updated components.

How to apply M3 Expressive
- Points to the motion physics guide and a session on building with M3 Expressive.
- Links to blog posts for a full introduction to the update and research insights.

How to use
- Use this file as a quick index for the official expressive guidelines entry point.
- Follow the linked style/component pages for detailed implementation guidance.
