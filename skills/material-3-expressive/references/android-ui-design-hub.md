# Android UI Design Hub

Source URL:
https://developer.android.com/design/ui/mobile

Scope
Android Developers hub for mobile UI design. Provides an overview plus links to guides, gallery, samples, and design kits. Emphasizes building mobile UI with Android themes, components, and platform patterns.

Key points
- Presents a mobile design overview and a guided entry into Foundations, Styles, Layout & content, Behaviors & patterns, Components, Home screen, Glossary, and Accessibility guides.
- Points to the Android design gallery for inspiration across screen sizes and app categories.
- Links to Android UI kit, Android Design Figma community, and the Material theme builder.
- Highlights window size classes to support compact, medium, and expanded layouts.

How to use
- Open this when you need official Android UI guidance that should remain consistent with Material 3 Expressive work.
- Use the guide sections to align expressive layouts with platform patterns (navigation, layout, behavior).
- Use the gallery and UI kit links for faster prototyping and visual validation.
