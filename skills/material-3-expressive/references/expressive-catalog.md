# Material 3 Expressive Catalog (GitHub)

Source URL:
https://github.com/meticha/material-3-expressive-catalog

Scope
Open source Jetpack Compose catalog demonstrating Material 3 Expressive components and patterns in a runnable app.

Key points
- Provides a catalog app for Material 3 Expressive UI components and examples.
- Explicitly lists examples for button groups, progress indicators, bottom app bar, FAB menu, and floating toolbar.
- Useful for inspecting visual behavior, layout rhythm, and component emphasis in Compose.

How to use
- Use this repo as a reference for how components look and how they are arranged in screens.
- Use it to sanity check your hierarchy, spacing, and expressive emphasis in Compose layouts.
